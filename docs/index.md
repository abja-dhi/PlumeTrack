# Plume Track Manual

Plume Track is a package developed by the DHI M&C Americas team to support the Singapore EMMP work with their analysis. It inlucdes various features to process collected data in the field including data gathered by vessel mounted ADCP and OBS vertical profiler. It also has tools to compare observed data with the MIKE HD and MT simulation results. This documentation includes two main sections:

1. **[User Guide](user_guide/0_installation.md)**: Step-by-step instructions on how to use Plume Track for various tasks.
2. **[Technical Reference](technical_docs/architecture.md)**: In-depth explanations of the algorithms and data structures used in Plume Track.
